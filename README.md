# Trixeather-Smart-Farming - Smart Irrigation v1

Sistem ini dibuat untuk memantau suhu tanaman secara otomatis.

### Logika yang Digunakan:
- Jika suhu > 30°C: Siram maksimal.
- Jika suhu != 25°C: Siram normal.
- Selain itu: Standby.

*Project ini adalah bagian dari portofolio menuju S1 Teknik Informatika.*
